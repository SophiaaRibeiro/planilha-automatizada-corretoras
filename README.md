# 📊 Planilha Automatizada - Corretoras de Sucesso

Projeto desenvolvido com foco na automação de processos e organização de atividades utilizando Excel e VBA (macros).

## 🚀 Objetivo

Criar uma solução prática para otimizar o controle de tarefas, melhorar a produtividade e facilitar o acompanhamento das atividades no dia a dia.

## ⚙️ Funcionalidades

- Automação de processos com macros (VBA)
- Controle de atividades e tarefas
- Organização baseada em metodologia ágil (Scrum)
- Otimização de tempo e redução de tarefas manuais

## 🛠️ Tecnologias utilizadas

- Microsoft Excel
- VBA (Visual Basic for Applications)

## 📌 Metodologia

O projeto foi estruturado utilizando conceitos de **Scrum**, com foco em:
- Organização de demandas
- Priorização de tarefas
- Acompanhamento contínuo das entregas

## 💡 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de aplicar conceitos de automação e organização, trazendo uma solução simples e eficiente para o gerenciamento de atividades.
